# 鼠标星星特效组件

## 功能特点
- 鼠标移动时生成星星效果
- 星星会跟随鼠标移动并逐渐消失
- 独立组件，不影响任何现有功能
- 可自定义星星大小、频率和持续时间

## 使用方法

### 1. 导入组件
```jsx
import MouseStars from './MouseStars'
```

### 2. 在应用中使用
```jsx
function App() {
  return (
    <div className="app">
      {/* 你的其他组件 */}
      <MouseStars />
    </div>
  )
}
```

### 3. 自定义配置（可选）
你可以通过修改组件文件来自定义：
- `star.size`: 星星大小范围（默认：10-30px）
- `star.delay`: 动画延迟（默认：0-0.5秒）
- 生成频率：修改 `100` 毫秒值（默认每100ms生成一个星星）
- 持续时间：修改 `3000` 毫秒值（默认3秒后移除）

## 样式自定义
可以通过修改 `MouseStars.css` 文件来调整星星的外观：
- 背景颜色
- 阴影效果
- 大小范围

## 注意事项
- 组件使用 `pointer-events: none` 确保不会影响其他元素的交互
- 星星会自动清理，不会造成内存泄漏
- 适用于任何React应用，无需修改现有组件

## 兼容性
- React 16.8+（使用Hooks）
- 需要framer-motion库支持动画效果
